

# Portfolio Website for Ravi Prajapati

A modern, dark-themed personal portfolio website with smooth animations and responsive design.

## Design & Theme
- **Dark background** with blue/purple/teal gradient accents
- Smooth scroll navigation with a fixed navbar
- Subtle fade-in and scale animations throughout
- Fully responsive across mobile, tablet, and desktop

## Sections

### 1. Hero Section
- Large name display "Ravi Prajapati" with gradient text effect
- Subtitle: "Full Stack Developer | MERN Stack Enthusiast"
- Brief intro paragraph about being a BCA student
- Three CTA buttons: View Projects, Contact Me, Download Resume (links to `/public/resume.pdf`)

### 2. About Section
- Professional summary about passion for development, problem-solving, academic/personal projects, and career goals
